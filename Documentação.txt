REQUISITOS FUNCIONAIS

| ID   | Requisito                       | Descrição                                                                      |
| ---- | ------------------------------- | ------------------------------------------------------------------------------ |
| RF01 | Cadastro de Usuário             | Permite que qualquer pessoa crie uma conta na plataforma.                      |
| RF02 | Login                           | Permite autenticação de usuários cadastrados.                                  |
| RF03 | Pesquisa de Filmes              | Permite pesquisar filmes cadastrados na plataforma sem necessidade de login.   |
| RF04 | Visualizar Informações do Filme | Permite visualizar os detalhes completos de um filme sem necessidade de login. |
| RF05 | Cadastro de Filme               | Permite que o administrador (admin@admin) cadastre novos filmes.               |
| RF06 | Editar Filme                    | Permite que o administrador (admin@admin) altere informações de filmes.        |
| RF07 | Excluir Filme                   | Permite que o administrador (admin@admin) remova filmes da plataforma.         |
| RF08 | Editar Perfil                   | Permite que usuários alterem seu nome e email.                                 |
| RF09 | Excluir Conta                   | Permite que usuários removam sua própria conta da plataforma.                  |
| RF10 | Controle Administrativo         | Restringe cadastro, edição e exclusão de filmes apenas ao administrador.       |
